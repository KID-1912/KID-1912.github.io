# Vue

## v-model

```v-model```能够在表单元素上快速创建数据双向绑定(语法糖)，不同表单元素代理对应input、change事件；

如果用于组件component上，则默认添加名为value的prop值传递，以及监听内部input事件；

### 自定义model

允许在子元素内自定义model，自定义prop的使用与通信；

```js
export default {
    model: {prop: 'activeIndex', event: 'active' },
    props: ['form'],
    methods: {
        onConfirm(i) {
            this.$emit('active', i);
        }
    }
}
```

### 指令修饰符

```txt
v-model.trim
v-model.number
v-model.lazy
```
