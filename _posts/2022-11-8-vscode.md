# VSCode

## 控制台

vscode右下角控制台可查看所有插件运行状态与输出日志

## 快捷键

### 注释

`ctrl + / ` 单行注释

`shift + alt + a` 多行注释

`shift + alt + i `多行尾部插入

**koroFileHeader** 插件额外支持

`ctrl + window + i` 生成文件头

`ctrl + window + t` 生成函数参数注释

**函数/模块注释格式**

```js
/**

 * 函数/模块名 描述功能

 * @param {string} name 名字

 * @param {*} value 数值

 * @params {类型说明} 参数名称 参数描述

 */
```
