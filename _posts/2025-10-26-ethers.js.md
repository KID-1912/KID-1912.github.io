# ethers.js

[官方](https://docs.ethers.org/)

## ethers.Contract

## Transaction

### TransactionResponse

```js
const tx = await contractInstance.callETH(...); // 调用合约方法
{
  hash: string,           // 交易哈希
  from: string,           // 发送者地址
  to: string,             // 接收者地址
  value: BigInt,          // 转账金额
  data: string,          // 调用数据
  gasLimit: BigInt,       // Gas 限制
  gasPrice?: BigInt,      // Gas 价格
  nonce: number,          // 随机数
  // ... 其他属性
}
```

### TransactionReceipt

```js
const receipt = await tx.wait();
{
  hash: string,           // 交易哈希
  blockNumber: number,    // 区块号
  blockHash: string,      // 区块哈希
  gasUsed: BigInt,        // 实际使用的 Gas
  gasPrice?: BigInt,      // Gas 价格
  status: number,         // 状态 (1=成功, 0=失败)
  logs: Array,           // 事件日志
  contractAddress: string | null, // 如果是部署交易，这里是合约地址
  // ... 其他属性
}
```
